# System Architecture

## Core Components
- **Backend**: Node.js services with modular controllers (`authController.js`, `appointmentController.js`).
- **Frontend**: React‑based patient and provider portals.
- **Database**: MongoDB with sharding and TLS.
- **Blockchain**: Smart contracts for medical record immutability.
- **CI/CD**: GitHub Actions (`ci.yml`, `main.yml`) with coverage and security scans.

## Design Principles
- Compliance‑first
- Modular and scalable
- Audit‑friendly logging
- Institutional readiness
