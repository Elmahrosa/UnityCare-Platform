## Pilot Overview

The UCH Institutional Pilot is a structured 90-day deployment designed to give procurement teams, clinical leadership, and IT governance a low-risk entry point to validate the platform before full institutional commitment.

**Pilot fee: $45,000 — fully credited toward any upgrade tier.**

---

## Pilot Scope

| Component | Included |
|---|---|
| Single-facility deployment | ✅ |
| White-label branding (institution name) | ✅ |
| Core modules: auth, patients, appointments, records | ✅ |
| Audit logging and GDPR workflows | ✅ |
| Role configuration (up to 5 roles) | ✅ |
| Onboarding and configuration support | ✅ |
| Weekly check-in calls | ✅ |
| Diligence documentation package | ✅ |

## Pilot Timeline

```
Week 1–2:   Infrastructure setup + configuration
Week 3:     User onboarding + role configuration
Week 4–8:   Active operational use
Week 9–10:  Performance review + feedback collection
Week 11–12: Conversion discussion + upgrade scoping
```

---

## Success Metrics

A pilot is considered successful when the following are met:

| Metric | Target |
|---|---|
| Platform uptime | ≥ 95% |
| Verified clinician usage | Active use by ≥ 3 roles |
| Critical security incidents | 0 |
| Documented operational improvement | At least 1 measurable outcome |
| Data ownership verified | Institution confirms full control |
| Executive intent to expand | Signed letter of intent or verbal commitment |

---

## What Happens After the Pilot

Upon successful completion:

1. **Upgrade decision** — institution selects full tier (License, IP Transfer, or Sovereign)
2. **$45,000 credited** in full toward the selected tier
3. **Full deployment scoped** — facilities, integrations, data residency, regulatory packaging
4. **Contract signed** → deployment begins within 30 days

---

## Pilot Exclusions

The following are not included in the pilot tier and are available in upgrade tiers:

- Multi-facility deployment
- Full IP transfer / source code handover
- Air-gapped / offline deployment
- Custom integrations (EHR, insurance APIs, IoT)
- 24/7 enterprise support SLA
