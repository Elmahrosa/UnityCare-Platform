# Procurement Model
**Unity Care Hospital (UCH)**  
*Confidential — Elmahrosa International*

---

## Overview

UCH is contracted as an institutional deployment — not a subscription. Pricing is engagement-scoped based on number of facilities, integrations required, hosting model, data residency requirements, and regulatory packaging.

**Pilot fees are fully credited toward any upgrade tier upon conversion.**

---
