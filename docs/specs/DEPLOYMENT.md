# Data Privacy

Unity Care Hospital handles sensitive healthcare data and therefore follows strict data privacy principles.

---

## Types of Data Managed

The system may process the following data categories:

- patient identification data
- medical consultation records
- prescription data
- appointment history
- billing information

---

## Data Ownership

All patient data remains the property of the healthcare institution operating the system.

The platform functions only as the processing infrastructure.

---

## Data Access

Access to patient data is controlled through role based permissions.

Typical access patterns include:

- doctors accessing patient consultations
- pharmacy staff accessing prescriptions
- administrators managing system records

Unauthorized access is restricted.

---

## Data Storage

Healthcare records are stored within the infrastructure environment selected by the deploying organization.

Possible environments include:

- hospital private servers
- government data centers
- secure cloud environments

---

## Data Retention

Data retention policies are defined by the healthcare organization using the system and may depend on national healthcare regulations.

---

## Compliance

The platform architecture supports privacy principles aligned with:

- GDPR
- HIPAA
- national healthcare data regulations

Actual compliance certification depends on the hosting environment and operational policies.
